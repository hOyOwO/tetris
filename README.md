## 📌 개요

Claude AI 활용으로 구현한 현대적인 테트리스입니다.
- ✨ 반응형 디자인 (데스크톱 + 모바일 완벽 지원)

- 📊 레벨 & 점수 시스템
- 🔄 지속적인 기능 개선

## 🎮 어떻게 플레이하나요?

### **데스크톱**
| 키 | 동작 |
|-------|--------|
| **←→** (화살표) | 좌우 이동 |
| **↑** (위 화살표) | 회전 (반시계 방향) |
| **↓** (아래 화살표) | 내리기 (소프트드롭) |
| **Space** | 즉시 낙하 (하드드롭) |
| **P** | 일시정지/재개 |

### **모바일 (터치)**
- **드래그** 좌우: 블록 좌우 이동
- **탭**: 블록 회전
- **위→아래 빠른 스와이프**: 즉시 낙하
- **천천히 아래로 드래그**: 소프트드롭
- **화면 아래 버튼**: 방향/회전/낙하/일시정지

## 🚀 어떻게 실행하나요?

### 온라인 (가장 간단함)
브라우저에서 바로 접속하세요:
```
https://hoyowo.github.io/tetris
```

### 로컬 환경에서 실행
1. **저장소 클론**
   ```bash
   git clone https://github.com/hOyOwO/tetris.git
   cd tetris
   ```

2. **로컬 서버 시작** (Python)
   ```bash
   # Python 3.x
   python -m http.server 8000
   ```
   또는
   ```bash
   # Python 2.x
   python -m SimpleHTTPServer 8000
   ```

3. **브라우저에서 열기**
   ```
   http://localhost:8000
   ```

### Node.js 사용 (대안)
```bash
npx http-server .
# http://localhost:8080 에서 접속
```

## 🎯 게임 시스템

### 점수 계산
- **1줄 완성**: 100 × 현재 레벨
- **2줄 완성**: 300 × 현재 레벨
- **3줄 완성**: 500 × 현재 레벨
- **4줄 완성 (테트리스!)**: 800 × 현재 레벨

### 난이도 진행
- 10줄 클리어마다 **레벨 1 증가**
- 레벨이 높아질수록 **낙하 속도 증가**

### 락 딜레이 (Lock Delay)
블록이 바닥에 닿은 후 **500ms 동안** 이동·회전 가능 (테트리스 표준)
- 이동/회전 성공 시 락 타이머 리셋
- 최대 15회까지만 리셋 (무한 회피 방지)
- 즉시 낙하 시 락 딜레이 없이 바로 고정

### 벽킥 (Wall Kick)
회전 시 벽에 충돌해도 ±1, ±2칸 오프셋으로 자동 재시도

## 🏆 리더보드

게임 오버 후 **닉네임을 입력하면 자동으로 TOP 10 리더보드에 등록**됩니다.
- 🥇 1위: 금메달
- 🥈 2위: 은메달
- 🥉 3위: 동메달
- 4~10위: 순위 표시

**버전 컬럼**: 각 기록이 어느 버전에서 나온 것인지 표시 (버전별 난이도 비교 가능)

## 📋 기술 스택

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript
- **렌더링**: Canvas API (2D)
- **게임 루프**: requestAnimationFrame (60fps)
- **배포**: GitHub Pages
- **리더보드 백엔드**: GitHub Contents API

### 파일 구조
```
tetris/
├── index.html          # 메인 게임 파일 (HTML + CSS + JS 올인원)
├── leaderboard.json    # 리더보드 데이터 (GitHub에 저장)
└── README.md           # 이 파일
```
